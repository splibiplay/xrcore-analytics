Cross-module telemetry and KPI module for XRCore ecosystems.

## Value in 2 Minutes

1. Install SDK + extension modules + Analytics.
2. Add Analytics runtime bridge to your scene.
3. Trigger events from Voice, VisionPlus, or LLBridge.
4. Confirm normalized analytics events and counters.

## What It Adds

- analytics event contracts and payload schemas
- profile-based telemetry configuration
- runtime subscribers to extension module events
- unified metrics output for release and QA workflows

## Ecosystem Position

```text

          ↓

Operational visibility and product KPIs
```
